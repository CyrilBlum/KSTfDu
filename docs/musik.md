# Musik

Programme für den Musikunterricht. Einige Programme sind lizenzpflichtig oder nur für Lehrpersonen vorgesehen.

## Ansprechpersonen

| Programm | Ansprechperson |
|---|---|
| MuseScore, BandLab, GarageBand, Ableton, Logic Pro, Dorico Pro, Sibelius | Stephan Pestalozzi, Silvia Strebel |

---

## MuseScore

MuseScore ist ein kostenloses Notationsprogramm für Schülerinnen.

### macOS

```bash
brew install --cask musescore
```

### Windows

```powershell
winget install -e --id Musescore.Musescore --scope machine --silent --accept-package-agreements --accept-source-agreements
```

---

## BandLab

BandLab ist eine kostenlose, einfach zu bedienende Sequenzer-Software als Webanwendung.

**Keine Installation nötig** – BandLab wird im Webbrowser geöffnet: [bandlab.com](https://www.bandlab.com)

---

## GarageBand *(nur macOS)*

GarageBand ist eine kostenlose Sequenzer-Software von Apple und steht nur auf macOS zur Verfügung. Die Installation erfolgt über das schulische MacBook-Klassensatz durch die IT.

**Schülerinnen mit privatem Mac:** GarageBand kostenlos im App Store installieren.

---

## Ableton

Ableton ist eine professionelle, kostenpflichtige Sequenzer- und Drummaschinen-Software. Die Lizenz wird durch die Schule bereitgestellt. Installation nur auf dem schulischen MacBook-Klassensatz durch die IT.

Download unter: [ableton.com](https://www.ableton.com)

---

## Logic Pro *(nur macOS, Klassensatz)*

Logic Pro ist eine professionelle Sequenzer-Software von Apple für macOS. Die Installation erfolgt auf dem schulischen MacBook-Klassensatz durch die IT.

---

## Dorico Pro *(nur für Lehrpersonen)*

Dorico Pro ist ein professionelles, kostenpflichtiges Notationsprogramm. Es wird ausschliesslich von Lehrpersonen verwendet.

### macOS und Windows

Download unter: [steinberg.net/dorico](https://www.steinberg.net/dorico/)

---

## Sibelius *(nur macOS, nur für Lehrpersonen)*

Sibelius ist ein professionelles, kostenpflichtiges Notationsprogramm für Lehrpersonen.

Download unter: [avid.com/sibelius](https://www.avid.com/sibelius)
