# Musik

Programme für den Musikunterricht. Einige Programme sind lizenzpflichtig oder nur für Lehrpersonen vorgesehen.

## Ansprechpersonen

| Programm | Ansprechperson |
|---|---|
| MuseScore, BandLab, GarageBand, Ableton, Logic Pro, Dorico Pro, Sibelius | Stephan Pestalozzi, Silvia Strebel |

---

## MuseScore

MuseScore ist ein kostenloses Notationsprogramm für Schülerinnen.

### macOS

```bash
brew install --cask musescore
```

### Windows

```powershell
winget install -e --id Musescore.Musescore --scope machine --silent --accept-package-agreements --accept-source-agreements
```

---

## BandLab

BandLab ist eine kostenlose, einfach zu bedienende Sequenzer-Software als Webanwendung.

### macOS

**Keine Installation nötig** – BandLab wird im Webbrowser geöffnet: [bandlab.com](https://www.bandlab.com)

### Windows

**Keine Installation nötig** – BandLab wird im Webbrowser geöffnet: [bandlab.com](https://www.bandlab.com)

---

## GarageBand *(nur macOS)*

GarageBand ist eine kostenlose Sequenzer-Software von Apple und steht nur auf macOS zur Verfügung.

### macOS

Die Installation erfolgt über das schulische MacBook-Klassensatz durch die IT.

**Schülerinnen mit privatem Mac:** GarageBand kostenlos im App Store installieren.

### Windows

GarageBand ist nicht für Windows verfügbar. Als Alternative steht [BandLab](https://www.bandlab.com) im Browser zur Verfügung.

---

## Ableton

Ableton ist eine professionelle, kostenpflichtige Sequenzer- und Drummaschinen-Software. Die Lizenz wird durch die Schule bereitgestellt.

### macOS

Installation nur auf dem schulischen MacBook-Klassensatz durch die IT.

Download unter: [ableton.com](https://www.ableton.com)

### Windows

Ableton ist für Windows verfügbar, wird aber an dieser Schule ausschliesslich auf dem schulischen MacBook-Klassensatz eingesetzt. Windows-Nutzerinnen wenden sich an die Ansprechpersonen.

---
