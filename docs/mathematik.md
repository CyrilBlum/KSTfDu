## GeoGebra Calculator Suite

GeoGebra ist eine kostenlose Geometrie-App mit integriertem Rechner. Sie steht als Webanwendung, Desktop-App und Tablet-App zur Verfügung.

**Web:** [geogebra.org/calculator](https://www.geogebra.org/calculator) – keine Installation nötig.

### macOS

GeoGebra ist nicht über Homebrew verfügbar. Download der Desktop-App unter: [geogebra.org/download](https://www.geogebra.org/download)

### Windows

```powershell
winget install -e --id GeoGebra.GeoGebra.Classic.6 --scope machine --silent --accept-package-agreements --accept-source-agreements
```
